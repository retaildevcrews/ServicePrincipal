# Service Principal Automation 

![License](https://img.shields.io/badge/license-MIT-green.svg)


## Summary
This solution is to automate the process of retrieving specific AAD objects, tracking changes, and enforcing policies through validation objects.  The initial implementation is focused on using Service Principals and ensuring that Business Owner email addresses are stored in the `Notes` field of the Service Principal object.
